1.图片太大显示不全 给 `<img>` 标签添加 `style="width: 160px; height: 160px; object-fit: contain;"` 限制尺寸并保持比例
2.布局对齐混乱 使用 `* { margin: 0; padding: 0; box-sizing: border-box; }` 重置浏览器默认样式，配合 Flex 布局实现精准对齐
3。Logo 显示变形 使用 `background: url("链接") no-repeat center; background-size: contain;` 让背景图自适应容器并居中显示
